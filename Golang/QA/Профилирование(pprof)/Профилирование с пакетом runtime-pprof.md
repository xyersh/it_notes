
Пакет `runtime/pprof` в языке Go предоставляет инструменты для профилирования производительности приложений. Он позволяет собирать данные о работе программы во время выполнения и анализировать узкие места, такие как потребление CPU, использование памяти, блокировки и другие.

## Импорты
`import "runtime/pprof"`

## Типы профилирования в `runtime/pprof`

| Тип профиля    | Описание                             |
| -------------- | ------------------------------------ |
| `cpu`          | Профилирование нагрузки CPU          |
| `heap`         | Распределение и использование памяти |
| `goroutine`    | Состояние и количество горутин       |
| `block`        | Время ожидания блокировок            |
| `mutex`        | Конфликты при работе с мьютексами    |
| `threadcreate` | Создание системных потоков           |
| `allocs`       | Все аллокации (включая собранные)    |

### Профилирование CPU

`pprof.StartCPUProfile(f io.Writer)` - запуск профилирования по CPU
`pprof.StopCPUProfile()` - остановка профилирования по CPU


### Профилирование использования памяти

`pprof.WriteHeapProfile(f io.Writer)` - запуск профилирования памяти
