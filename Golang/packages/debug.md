
- **debug.SetMaxThreads(n nThreads) defaultVal int** - Устанавливает максимально допустимое количество потоков. Возвращает предыдущий лимит количества потоков. 
- **debug.SetGCPercent(value)** - устанавливает рост значения размера кучи от старого размера в процентном соотношении , при котором будет включаться `GC`. Если `value = -1`, то GC на рост размера кучи не смотрит
- **debug.SetMemoryLimit(heap_value)** - устанавливает размер кучи в байтах, при котором будет запускаться GC
