

### NUMBER to INTERVAL

#### `NUMTODSINTERVAL` (Дни, часы, минуты, секунды)


`NUMTODSINTERVAL(n, 'единица_измерения')`
Где `'единица_измерения'` может быть
- `'DAY'` (День)
- `'HOUR'` (Час)
- `'MINUTE'` (Минута)
- `'SECOND'` (Секунда)


**Преобразование 10.5 дней:**
```sql
SELECT NUMTODSINTERVAL(10.5, 'DAY') FROM dual; 
-- Результат: +000000010 12:00:00.000000
```


**Преобразование 90 минут:**
```sql
SELECT NUMTODSINTERVAL(90, 'MINUTE') FROM dual;
-- Результат: +000000000 01:30:00.000000
```

**Использование с датой (например, прибавить 45 минут к текущему времени):**
```sql
SELECT SYSDATE + NUMTODSINTERVAL(45, 'MINUTE') AS future_time FROM dual;
```


#### `NUMTOYMINTERVAL` (Годы и месяцы)


```sql
NUMTOYMINTERVAL(n, 'единица_измерения')
```
Где `'единица_измерения'` может быть:
- `'YEAR'` (Год)
- `'MONTH'` (Месяц)

Преобразование 5 лет:
```sql
SELECT NUMTOYMINTERVAL(5, 'YEAR') FROM dual;
-- Результат: +000000005-00
```


Преобразование 18 месяцев:
```sql
SELECT NUMTOYMINTERVAL(18, 'MONTH') FROM dual;
-- Результат: +000000001-06
```

Использование с датой (например, отнять 3 месяца от текущей даты):
```sql
SELECT SYSDATE - NUMTOYMINTERVAL(3, 'MONTH') AS past_time FROM dual;
```
