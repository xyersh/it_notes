**.dockerignore** - использоется для исключения ненужных файлов из контекста сборки образа
Символы для шаблона:
- `*` - любое количество символов
- `?` - один символ
- `**` - поддиректории. Пример `**/node_modules`
- `#` - коментарий
- `!` - инвертирование правила (включение файла, если он вклчен выше)
